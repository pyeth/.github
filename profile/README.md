## What This Is

PYETH is the Python-Ethereum bridge — 83 repositories collecting every major Python tool for interacting with the Ethereum Virtual Machine. From web3.py to Brownie to Vyper to Ape, this is the complete Python EVM development toolkit.

## Domain

**Python + EVM:** web3.py, Brownie, Vyper, Ape, Python smart contract tools, EVM interaction, transaction building, contract deployment, testing frameworks, Python DeFi tooling
